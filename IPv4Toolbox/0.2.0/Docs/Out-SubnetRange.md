# Out-SubnetRange

## SYNOPSIS
Outputs a list of IPv4 Addresses from a CIDR address range.

## SYNTAX

### Prefix (Default)
```
Out-SubnetRange [-Subnet] <String> [-Prefix <Int32>] [<CommonParameters>]
```

### SubnetMask
```
Out-SubnetRange [-Subnet] <String> -SubnetMask <String> [<CommonParameters>]
```

## DESCRIPTION
Outputs a list of IPv4 Addresses from a CIDR address range.

## EXAMPLES

### EXAMPLE 1
```powershell
Out-SubnetRange -Subnet 192.168.1.0 -Prefix 28

192.168.1.1
192.168.1.2
192.168.1.3
192.168.1.4
192.168.1.5
192.168.1.6
192.168.1.7
192.168.1.8
192.168.1.9
192.168.1.10
192.168.1.11
192.168.1.12
192.168.1.13
192.168.1.14
```
