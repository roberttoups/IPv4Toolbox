# Split-Subnet

## SYNOPSIS
Breaks up a larger CIDR into small CIDRs.

## SYNTAX

```
Split-Subnet [-Subnet] <String> [-Prefix] <Int32> [-TargetPrefix] <Int32> [<CommonParameters>]
```

## DESCRIPTION
Breaks up a larger CIDR into small CIDRs.

## EXAMPLES

### EXAMPLE 1
```
Split-Subnet -Subnet 10.2.2.0 -Prefix 24 -TargetPrefix 25
```

SubnetId            : 10.2.2.0
BroadcastAddress    : 10.2.2.127
